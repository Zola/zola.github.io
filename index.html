AutoHan – 自动简繁转换

汉字[Kanji Hanzi]不止汉族在使用，在的其他种族也在使用汉字。汉字写法多样，导致各地汉语使用者有不同汉字阅读习惯。AutoHan 项目是一段可嵌入到网页的 JavaScript 代码，能根据浏览器预设接受的语言类型自动切换至使用者偏好的繁体汉字或简体汉字。

本插件有不同的应用场景：
  1. 在 Chrome 的网上应用店搜索 AutoHan 插件即可自动安装，也可下载后手动安装。即可手动将正在浏览的网页切换至繁体汉字界面或简体汉字界面，从而加快阅读速度。
  2. 在 WordPress 中搜索 AutoHan 插件即可自动安装，也可下载后手动安装。即可让你的网站自动切换至使用者偏好的繁体汉字或简体汉字。
  3. 如果你在制作网页，在网页尾部嵌入以下代码，即可像本站一样自动切换至使用者偏好的繁体汉字或简体汉字。 
    <script src="https://www.autohan.org/js/Std_StranJF.js" integrity="sha384-dCV+UmPZoyu+EQKH2Feh0Omt4XRyIZT8QOsbFvLIr022OVZg9WmgIctnlHCnEJiX" crossorigin="anonymous"></script>

    手动切换代码：
    <a href='javascript:AutoHan_StranBody()' id='StranLink'>繁体界面</a>

 

为何需要转换

有的人阅读繁体字比较吃力，有的人阅读简体字比较吃力。AutoHan 自动繁简转换项目致力于帮助阅读汉字的使用者。

GB2312 编码的字库收录 6763 个汉字，但不能显示繁体字；港澳台等地用的 BIG5 编码收录 13060 个汉字，但不能显示简体字。现在普遍的是 UTF-8 编码，能同时显示繁体和简体。

AutoHan 是一段能自动将 1275 个简体字及繁体字进行对应转换的 JavaScript 代码，可以用于 WordPress 、 Chrome 和任意汉字网站，帮助你或你的网站访客提高阅读速度。可成为民宿、电商、个人网站的简易的多语言界面解决方案，提高客户转化率，且不會減慢您的網站速度。

注意，AutoHan 并不负责把“贾伯斯”转换为“乔布斯”，也不会把“賓士”转换成“奔驰”，不会把“汉人”转换为“华人”，不会转换“台湾”为“中华台北”。

运作方式介绍

透过辨识浏览器预设优先支援的语言类型（你现在的浏览器语言是），来决定作何种转换。习惯繁体汉字的读者看到的是繁体内容，习惯简体汉字的人看到的是简体内容。 还能借助浏览器 cookies 来记住你偏好的语言的设定。使用者可以变更 Google Chrome 浏览器的语言优先顺序，测试这个网站在不同地区的语言的浏览器里的不同表现。
