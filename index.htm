AutoHan – 自動簡繁轉換

漢字[Kanji Hanzi]不止漢族在使用，在的其他種族也在使用漢字。漢字寫法多樣，導致各地漢語使用者有不同漢字閱讀習慣。AutoHan 專案是一段可嵌入到網頁的 JavaScript 程式碼，能根據瀏覽器預設接受的語言型別自動切換至使用者偏好的繁體漢字或簡體漢字。

本外掛有不同的應用場景：
  1. 在 Chrome 的網上應用店搜尋 AutoHan 外掛即可自動安裝，也可下載後手動安裝。即可手動將正在瀏覽的網頁切換至繁體漢字介面或簡體漢字介面，從而加快閱讀速度。
  2. 在 WordPress 中搜索 AutoHan 外掛即可自動安裝，也可下載後手動安裝。即可讓你的網站自動切換至使用者偏好的繁體漢字或簡體漢字。
  3. 如果你在制作網頁，在網頁尾部嵌入以下程式碼，即可像本站一樣自動切換至使用者偏好的繁體漢字或簡體漢字。 
    <script src="https://www.autohan.org/js/Std_StranJF.js" integrity="sha384-dCV+UmPZoyu+EQKH2Feh0Omt4XRyIZT8QOsbFvLIr022OVZg9WmgIctnlHCnEJiX" crossorigin="anonymous"></script>

    手動切換程式碼：
    <a href='javascript:AutoHan_StranBody()' id='StranLink'>繁體介面</a>

 

為何需要繁簡轉換？

文化背景

由於不同國家的文化背景的差異，部分讀者在閱讀未被簡化的繁體字（正體字）時會感到吃力，而另一些讀者則在閱讀簡化的簡體字和異體字時遇到困難。AutoHan 自動繁簡轉換專案專為解決這一問題設計，旨在幫助所有使用漢字的讀者更輕鬆地理解內容。

技術背景

早期的 GB2312 編碼支持 6763 個簡體漢字，但無法顯示繁體字；而港澳臺等地使用的 BIG5 編碼則支持 13060 個繁體漢字，但同樣無法顯示簡體字。如今，通用的 UTF-8 編碼能同時顯示繁體和簡體字，甚至支持 emoji 表情符號及其他新字符，但異體字轉換需求仍未被滿足。

我們的解決方案

AutoHan 是一款 JavaScript 工具，能自動對應轉換 1275 個簡體字與繁體字。它適用於 WordPress 、 Chrome 擴展 以及任何漢字網站，有助於提升個人或網站訪客的閱讀效率。這款工具特別適合用於民宿電商、個人網站等場景，為用戶提供簡易的多語言介面解決方案，進一步提升客戶轉化率，同時不影響網站的加載速度。

重要提醒

AutoHan 不支持詞語級的地域用詞轉換，無法將「賓士」轉換成「奔馳」，或將「賈伯斯」轉換為「喬布斯」。若需地域用詞轉換，可使用「線上繁簡轉換」工具體驗 OpenCC 的地域用詞轉換功能。若需精確的出版級轉換，建議嘗試帶有自定義轉換詞典功能的付費版，可以實現「中華臺北」轉換為「臺灣」，「華人」轉換為「漢人」。

運作方式介紹

透過辨識瀏覽器預設優先支援的語言型別（你現在的瀏覽器語言是），來決定作何種轉換。習慣繁體漢字的讀者看到的是繁體內容，習慣簡體漢字的人看到的是簡體內容。 還能借助瀏覽器 cookies 來記住你偏好的語言的設定。使用者可以變更 Google Chrome 瀏覽器的語言優先順序，測試這個網站在不同地區的語言的瀏覽器裡的不同表現。
